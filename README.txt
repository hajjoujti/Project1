		|							|
		---------------------------------------------------------


	-------------------------- Première fenêtre -----------------------------

	--> Première utilisation du logiciel :

	    > Appuyer sur le bouton "Nouveau" et sélection le fichier .txt que vous
	      souhaitez utiliser.
	    > Une fois créé cliquez sur le bouton Accueil pour pouvoir accèder aux
	       fonctionnalités de l'application.
	  
	-------------------------- Fenêtre utilisateur ----------------------------- 

	   --> Ajouter un stagiaire :

	       > Appuyez sur le bouton "+" et insérez les coordonnées du stagiaire à 
		 ajouter dans les champs indiqués.
	       > Appuyez sur ajouter pour valider. 

	   --> Recherche d'un stagiaire dans la liste :

	       > Dans la colonne stagiaire, indiquez au minimum un élément à partir 
		duquel vous voulez effectuer votre recherche.
	       > Cliquez sur le bouton "Rechercher" afin d'afficher le résultat de votre
	        recherche sur le tableau.
	       > Cliquez sur réinitialiser pour afficher le tableau d'origine. 

           --> Exportation du tableau au format PDF : 
	       > Cliquez sur le bouton PDF pour télécharger le tableau d'origine ou le
		 résultat de votre recherche au format .pdf
		
	   --> Document d'utilisation :
    	       > Cliquez sur le bouton "télécharger la doc" pour afficher le fichier 
		'README' ci présent.
		
	   --> Connection en tant qu'administrateur : 
	       > Cliquez sur le bouton "Session Administration" pour pouvoir vous connecter
		en tant qu'administrateur.
	       > Entrez vos identifiants et mots de passe pour valider l'accès.


 	-------------------------- Fenêtre Administrateur -----------------------------	

	   **En tant qu'administrateur, vous disposez de toutes les fonctionnalitées 
	      utilisateur**

	   --> Supprimer un stagiaire de l'annuaire :
	       > Cliquez sur l'icone POUBELLE rouge au niveau du stagiaire que vous 
		 vous souhaitez supprimer.
	       	
	   --> Modifier un stagiaire de l'annuaire :
	       > Cliquez sur l'icône CRAYON pour faire apparaître la fenêtre de modification.
	       > Saisir les modification dans les champs destiné.
